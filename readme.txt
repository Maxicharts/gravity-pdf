== Description ==

Extends MaxiCharts plugin integrating with [Gravity PDF](https://gravitypdf.com/ "Gravity PDF") plugin and allowing graphs insertions inside PDF notification emails. This plugin embeds powerfull CasperJS and PhantomJS librairies and has thus quite a big size.

### Usage ###

Simply insert any Maxicharts shortcode inside a hidden field, where you want the graph to appear.

== Installation ==

### Easy ###
1. Search via plugins > add new.
2. Find the plugin listed and click activate.
3. Use the Shortcode
